 # ຕົວຢ່າງການປະຍຸກໃຊ້
   ສາມາດໃຊ້ກວດຈັບຄົນຮ້າຍ,ບຸກຄົນທີຕ້ອງການຊອກຫາໄດ້ ຫລື ກວດຈັບພະນັກງານເຂົ້າອອກ office. ເພີ່ມເຕີມແມ່ນເພີ່ມ Ultrasonic Sensor ກວດຈັບວັດຖຸທີແລນຜ່ານກ້ອງແລ້ວໃຫ້ກ້ອງເຮັດວຽກ ເພື່ອເພີ່ມປະສິດທິພາບໃຫ້ລະບົບຫລາຍຂຶ້ນ.
